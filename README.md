# loto-game — игра ЛОТО

Одностраничное офлайн-приложение для проведения ЛОТО на мероприятии. У каждого
участника на бейдже 5 чисел (по одному из 5 разных десятков). Ведущий вытаскивает
бочонки 1–99; участник побеждает, когда закрыты все 5 его чисел. Игра идёт, пока
не наберётся заданное число победителей.

Один файл `index.html` — без внешних библиотек, всё работает в браузере, данные
не покидают устройство.

## Вкладка «Участники»
- Вставка списка из Excel/1С или загрузка CSV (авто-определение разделителя и
  кодировки UTF-8 / Windows-1251).
- Форматы строки: `Имя, Фамилия, Компания, Должность` (числа сгенерируем) или
  `Имя, Фамилия, Компания, Должность, Ч1, Ч2, Ч3, Ч4, Ч5` (числа заданы).
  Можно смешивать; поля с запятой внутри — в кавычках. Минимально достаточно
  `Имя, Фамилия`.
- Генерация уникальных комбинаций (5 чисел из 5 разных десятков).
- Экспорт CSV `Имя, Фамилия, Компания, Должность, Число1..5` для печати бейджей
  (проект `badge-print`).

## Вкладка «Игра»
- Настройка количества победителей.
- Вытаскивание бочонков 1–99, табло, история, прогресс участников.
- Объявление победителей при закрытии всех 5 чисел; при ничьей показываются все,
  закрывшиеся на одном бочонке.

## Тесты
Чистая логика покрыта тестами (Node ≥ 18). Из папки проекта:

    node --test "test/*.test.mjs"
